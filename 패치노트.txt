v1.1.0

1. 작업 내용을 임시저장할 수 없는 문제가 수정됩니다.

2. 프로그램 로그를 기록하는 기능을 추가했습니다.

 - 로그는 다음 내용만을 수집 및 기록합니다.
    1) 프로그램 실행 시각
    2) 사용한 GPU 장치 이름 및 용량
    3) 오류 발생 시 내역 및 발생 시각
   이 외에는 어떠한 정보도 수집하지 않으며, %AppData%\Oasis\logs에 날짜별로 기록됩니다.
 - 로그 파일은 컴퓨터에 직접 저장되며, 자동으로 전송되지 않습니다.

3. 프로그램이 비정상종료 되는 치명적인 오류 발생 시 종료 대신 메세지를 띄우도록 수정됩니다.

 - 메세지가 뜰 경우 작업 내용이 손상될 수 있으므로 즉시 작업을 저장하는 것을 권장합니다.
   로그파일을 보관 및 보고해주시면 빠르게 수정하도록 하겠습니다.

4. 단축키가 간헐적으로 작동하지 않는 문제가 완화됩니다.
 - 프로그램 내부를 클릭하거나 esc키를 누르면 포커싱이 위젯에서 빠져나오도록 개선했습니다.

5. 동물종 목록 시스템을 개선했습니다.

 - 드래그&드롭으로 동물종명의 위치를 편집할 수 있습니다.
 - 기본제공 동물 DB를 추가했습니다.
    (포유류 125종, 조류 545종, 파충류32종 합 702종)
 - Main 탭의 시트에서 국명을 직접 기록할 경우 DB에 존재하는 동물이면 자동으로 학명도 기입되는 기능을 추가했습니다,

6. 촬영된 카메라의 파일저장시각과 실제 촬영시각이 일치하지 않는 경우를 위해 시간 조절기능을 추가했습니다.

7. 밝기를 조절할 수 있는 기능을 추가했습니다.
 - RGB연산으로 인해 동영상 재생에서는 밝기가 0이 아닌 경우 소폭의 속도저하가 발생할 수 있습니다.

8. Result 탭의 스프레드 시트의 행을 한 번에 여러 개 지울 수 있도록 개선됩니다.

9. 실수를 방지하기 위해 스프레드 시트에 실행 취소 및 다시 실행 기능을 추가했습니다.

10. 인식된 동물이 없을 경우 국명, 학명 및 개체 수가 기본값으로 초기화 되지 않도록 개선됩니다.

11. 기록(구 테이블 저장)버튼을 누를 때 {기본파일명}_동물이름 으로 파일명 자동 변경 및 기록 기능을 추가했습니다.
 - Settings -> 프로그램 설정 -> 자동 이름 변경 체크 시 작동합니다. * 기본값 : 해제

12. Main 탭의 테이블데이터 수정을 클릭 또는 더블클릭으로 사용자가 선택할 수 있도록 개선했습니다.
 - Settings -> 프로그램 설정 -> 더블클릭으로 테이블 수정 * 기본값 : 체크

13. 다음 버튼을 누를 때 소폭의 딜레이를 추가했습니다.

14. 파일 제거 기능을 추가했습니다.
 - '제거'된 파일은 휴지통이 아닌 기존 실행 경로상의 trash 폴더로 이동됩니다.

15. 사진 및 영상의 줌인 & 줌아웃 기능을 추가했습니다.
 - 사진 위에서 마우스 휠을 통해 조절할 수 있습니다.

16. 원하는 파일로 이동하는 기능을 더블클릭 -> 클릭으로 변경하고, 강조를 위해 버튼형태로 변경됩니다.

17. 일부 단축키를 사용자가 직접 변경할 수 있도록 개선했습니다.



v1.0.2

1. 딥러닝 장치를 선택할 수 있도록 CPU 옵션을 추가했습니다.

2. GPU 호환이 되지 않을 경우에도 자동으로 CPU를 사용하도록 개선됩니다.

 - 기존에는 GPU 사용가능여부에 따라 판단했으나
   호환여부도 체크하도록 코드를 수정했습니다.

3. 프로그램 사용 도중 사진을 읽어올 수 없을 경우 더미파일이 생성되는 문제가 수정됩니다.



v1.0.1

1. 파일명 변경시 한국어를 무시하는 오류가 수정됩니다.

2. 일부 어색한 문구를 수정하였습니다.

3. 엑셀로 저장 시 특정 컴퓨터에서 저장에 실패하였습니다! 문구와 함께 저장되지 않는 오류가 수정됩니다.

4. 시트 저장 관련 프로세스를 최적화하였습니다.



v1.0.0

1. Oasis 프로그램을 정식으로 배포합니다!

2. Oasis 프로그램은 YoloV5 기반의 딥러닝 AI가 멧돼지와 고라니 및 산양의 판단을 보조해줍니다.

3. Oasis 프로그램은 사진 및 영상의 생성 날짜 및 시각을 자동으로 가져옵니다.

4. Oasis 프로그램은 사진의 칼라/흑백을 기준으로 주간/야간을 자동으로 판독해줍니다.
